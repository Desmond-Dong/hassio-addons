## 关于

[Node-RED][nodered] 是一个用于以新颖有趣的方式连接硬件设备、API 和在线服务的编程工具。

它提供了一个基于浏览器的编辑器，可以轻松地使用调色板中的各种节点连接流程，并且只需一次点击即可将其部署到其运行时。
