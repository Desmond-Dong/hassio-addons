图书管理器和自动化（用于电子书的Sonarr）。

## 关于

[Readarr] 是一个用于Usenet和BitTorrent用户的电子书和有声书收藏管理器。它可以监控多个RSS源以获取您喜欢的作者的最新书籍，并会抓取、排序和重命名它们。请注意，每种类型的书籍只支持一种格式。
