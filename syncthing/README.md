# Syncthing 插件

[**Syncthing**](https://syncthing.net/) 被包含在 [Home Assistant Supervisor 插件](https://www.home-assistant.io/addons/) 中。

> Syncthing 是一个持续文件同步程序。它实时地在两台或多台计算机之间同步文件，并安全地保护免受窥视。你的数据只属于你，你值得选择数据的存储位置、是否与第三方共享以及如何在互联网上传输数据。
