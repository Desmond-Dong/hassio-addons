# MPD

[MPD](https://www.musicpd.org/) 是一个免费且开源的音乐播放器。它可以播放音频文件、组织播放列表并维护音乐数据库。需要交互式客户端软件进行操作。
在这个插件中，包含有 [ympd](https://ympd.org/)，一个简单的用于交互的网页前端。
