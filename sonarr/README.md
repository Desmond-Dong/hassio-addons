智能PVR（个人视频录制）系统，适用于新sgate组和BitTorrent用户。

## 关于

[Sonarr] 是一个适用于新sgate组和BitTorrent用户的PVR系统。它可以监控多个RSS源，以获取您喜爱的节目的新剧集，并进行抓取、排序和重命名。它还可以配置为在更高质量的格式可用时自动升级已下载文件的质量。
