智能的 PVR 适用于新闻组和 Bittorrent 用户。

## 关于

[Sonarr] 是一个针对 Usenet 和 BitTorrent 用户的 PVR。它可以监控多个 RSS 源，以获取您喜欢的节目的新剧集，并会抓取、分类和重命名它们。它还可以配置为自动升级已下载文件的质量，当更好的格式可用时。
