# OWASP Juice Shop

[OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) 可能是现代且最复杂的易受攻击的Web应用程序！它可用于安全培训、意识演示、CTF，以及作为安全工具的实验鼠！Juice Shop 涵盖了OWASP Top Ten中的所有漏洞，以及许多在现实世界应用程序中发现的其它安全问题！
