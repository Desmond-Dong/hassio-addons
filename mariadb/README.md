# Home Assistant Add-on: MariaDB

MariaDB数据库用于Home Assistant。

![支持aarch64架构][aarch64-shield] ![支持amd64架构][amd64-shield] ![支持armhf架构][armhf-shield] ![支持armv7架构][armv7-shield] ![支持i386架构][i386-shield]

## 关于

您可以使用此插件来安装MariaDB，它是一个开源（GPLv2许可）的数据库。MariaDB可以用作Home Assistant的数据库后端。更多信息，请参见[MariaDB][mariadb]
