## 关于

[EMQX][emqx] 是一个开源的 MQTT 代理，具有高性能的实时消息处理引擎，能够为大规模物联网设备提供事件流服务。作为最具可扩展性的 MQTT 代理，EMQX 可以帮助您连接任何设备，无论规模如何（包括您的家庭）。

[EMQX MQTT 代理][emqx] 是 Home Assistant 中通常使用的 Mosquitto MQTT 代理/附加组件的高级替代方案。它具有用户界面，用于配置、管理和调试您的 MQTT 代理、客户端和流量。

虽然 EMQX 主要在其网站上以云托管产品出售，但此附加组件在完全本地的自托管环境中运行 EMQX。
