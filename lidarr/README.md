看起来像Sonarr，但专为音乐设计。

## 关于

[Lidarr] 是一个用于Usenet和BitTorrent用户的音乐集合管理器。它可以监控多个RSS源以获取您最喜欢的艺术家的最新曲目，并将其抓取、排序和重命名。它还可以配置为在更好的质量格式可用时自动升级已下载文件的质量。

[Lidarr]: https://lidarr.audio/
