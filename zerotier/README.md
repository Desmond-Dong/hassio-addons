## 关于

ZeroTier 通过单一系统提供 VPN、SDN 和 SD-WAN 的功能。管理您所有连接资源，无论本地还是广域网络，都像整个世界是一个单一的数据中心一样。

人们使用 ZeroTier 无缝连接笔记本电脑、台式机、手机、嵌入式设备、云资源和应用程序，无论他们想去哪里。它将整个世界变成一个单一的数据中心，您现在可以使用此插件添加您的 Home Assistant 实例。
