# 非官方 Home Assistant 插件：Loki

Loki (<https://grafana.com/oss/loki/>) 被打包为 Home Assistant 插件。

## 关于

Grafana Loki 是一个水平可扩展、高可用、多租户的日志聚合系统，灵感来自 Prometheus。它旨在以非常具有成本效益和易于操作的方式运行。它不索引日志的内容，而是为每个日志流设置一组标签。<https://grafana.com/oss/loki/>
