## 关于

FlexGet 是一个用于所有媒体的多功能自动化工具。查看 https://flexget.com/

默认密码是 'homeassistant123'，或可以从添加项选项中设置

## 要求

## 配置

    "PUID": 自定义用户（默认 root）
    "PGID": 自定义用户（默认 root）
    "WebuiPass": Webui 密码定义
    "FG_PLUGINS": 插件添加
    "FG_LOG_LEVEL": 日志级别（critical|error|warning|info|verbose|debug|trace）

## 启动

- 启动添加项。稍等片刻并检查日志中的任何错误。

## 故障排除
