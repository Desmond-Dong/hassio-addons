# Home Assistant 插件：stt-fallback

![支持 aarch64 架构][aarch64-shield] ![支持 amd64 架构][amd64-shield]

一个使用 [语音到文本实体](https://www.home-assistant.io/integrations/stt) 进行回退的 Home Assistant 插件。

需要 Home Assistant 2023.11 或更高版本。
