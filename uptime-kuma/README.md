## 关于

Uptime Kuma 是一个开源监控工具，可以最好地与商业服务 "Uptime Robot" 的自托管版本进行比较。

它能够让你监控 HTTP/S、TCP、DNS 和其他协议的服务，并且可以在发生停机时向你发送通知或触发 Home Assistant 的自动化网络钩子。
