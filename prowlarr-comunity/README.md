## 关于

[Prowlarr] 是一个基于流行的 *arr .net/reactjs 基础堆栈构建的索引器管理器/代理，用于与您的各种 PVR 应用集成。Prowlarr 支持管理 torrent 跟踪器和 Usenet 索引器。它无缝集成 Lidarr、Mylar3、Radarr、Readarr 和 Sonarr，为您提供完整的索引器管理，无需每个应用单独设置索引器（它全部搞定）。
