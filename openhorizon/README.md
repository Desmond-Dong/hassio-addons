# 🌟 - Open Horizon agent

[Open Horizon](http://github.com/open-horizon) agent作为附加组件

## 平台：

+ `amd64`
+ `armv7`
+ `aarch64`

## 更改日志与发布
发布基于语义版本控制，并使用`MAJOR.MINOR.PATCH`的格式。简而言之，版本将根据以下方式增加：

- `MAJOR`：不兼容或主要更改。
- `MINOR`：向后兼容的新功能和增强。
- `PATCH`：向后兼容的bug修复和包更新。
